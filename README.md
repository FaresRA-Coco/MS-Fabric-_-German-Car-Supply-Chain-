# 🚗 German Automotive Supply Chain Analytics Platform  
### *End‑to‑End Microsoft Fabric Project (Synthetic Data)*

I just completed my most ambitious data project yet — a full end‑to‑end supply chain analytics platform for the German automotive industry, built entirely on **Microsoft Fabric**.

📌 **All data is fully synthetic and generated exclusively for learning and demonstration purposes.**  
No real, confidential, or proprietary information was used.

---

## 🔧 Technology Stack

- **Eventstream → Bronze → Silver → Gold** (Medallion Architecture)
- **GBT Anomaly Detection Model** in PySpark  
  - AUC: **0.928**
- **Five Power BI Dashboards**  
  - Production  
  - Quality  
  - Supply Chain  
  - Vehicle Performance  
  - Executive Summary
- **Supplier Route Mapping** across Germany
- **Plant Performance Benchmarking** across five warehouses

---

## 💡 Key Insight

Across all simulated production lines, **assembly misalignment** emerged as the most frequent defect cause.  
A single operational improvement in this area could raise **First Pass Yield (FPY)** above target across all brands.

---

## 🎯 Project Purpose

This project was built to simulate how a real OEM might:

- Design a scalable data architecture  
- Integrate machine learning into operational analytics  
- Build unified dashboards for leadership  
- Transform raw events into business‑ready KPIs  

It highlights how much **storytelling** lives inside a well‑structured data pipeline — from ingestion to ML to visualization.
